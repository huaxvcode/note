# 13.给组件自定义属性

## 接收传入的属性

### 数组的形式一次性接收多个变量

```html
<div class="box">
  <p>{{ title }}</p>
  <p>{{ author }}</p>
  <p>{{ content }}</p>
</div>
```

```js
<script setup>
  /* 接收传入的 title、author、content 变量（属性） */
  defineProps(['title', 'author', 'content']);
</script>
```

### 对象的形式一次性接收多个变量

```html
<div class="box">
  <p>{{ title }}</p>
  <p>{{ author }}</p>
  <p>{{ content }}</p>
</div>
```

```js
<script setup>
  defineProps({
    'title': String,
    'author': String,
    'content': String
  });
</script>
```

## 一些命名细节

对于 `Props` 属性（变量），采用小驼峰命名法 `wordWrod`，但是在 html 传入属性时，使用 `word-word` 的形式（二者都能被识别成 `wordWrod`）
