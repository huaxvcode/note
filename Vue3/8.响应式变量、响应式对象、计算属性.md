# 8.响应式变量、响应式对象、计算属性

## ref

使用 ref 放回一个响应式对象，对象里面的 `value` 属性是响应式变量

```js
<script setup>
import { ref } from 'vue';
const cnt = ref(0);
// bgc 不是响应变量，因此无法实现颜色轮变的效果：
// let bgc = "green";

// bgc 是响应变量，可以实现颜色的轮变：
const bgc = ref('green');

const setBgc = () => {
  cnt.value ++;
  if (cnt.value % 2 == 0) {
    bgc.value = 'green';
  }
  else {
    bgc.value = 'red';
  }
}

</script>
```

```html
<div box @click="setBgc()" :bgc="bgc"></div>
```
