# admonition 告诫框

## 告诫框

```html
<div class="admonition note">
    <div class="title">
        <div class="p">note</div>
    </div>
    <div class="content">
        <div class="p">用来存放一些笔记</div>
    </div>
</div>
```

## 可折叠告诫框

```html
<div class="admonition note details">
    <div class="title">
        <div class="p">note</div>
    </div>
    <div class="content">
        <div class="p">用来存放一些笔记</div>
        <div class="admonition note details">
            <div class="title">
                <div class="p">note</div>
            </div>
            <div class="content">
                <div class="p">用来存放一些笔记</div>
            </div>
        </div>
    </div>
</div>
```
